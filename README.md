# Tournament Scoring System

## Introduction

The Tournament Scoring System is a robust application designed to facilitate the management and scoring of tournaments, accommodating both individual and team events. 
This application is built using Python with a Tkinter GUI, making it user-friendly and accessible for tournament organizers.

## Features

- **Tournament Creation**: Easily set up tournaments with detailed configurations including event types and scoring systems.
- **Player & Team Management**: Manage participant registrations and team compositions with ease.
- **Customizable Points System**: Make the scoring scheme compatible to the tournament's requirements, supporting various outcomes such as wins, draws, and losses.
- **Rankings & Results**: Automatically calculate and display rankings for individuals and teams based on performance in events.
- **User-Friendly Interface**: Designed with a straightforward GUI to streamline tournament management processes.

## Setup & Installation

Ensure you have Python 3.6 or later installed on your system. To get started with the Tournament Scoring System:

1. Clone the repository or download the ZIP to your computer.
2. Navigate to the project directory.
3. Run the application:
   ```bash
   python scoring_system.py

## Usage Guide

Upon launching the application, you'll encounter the main menu from which you can access all functionalities:

-Set Points System: Customize how points are allocated for various rankings and outcomes.
-Start New Tournament: Initiate the creation of a new tournament with custom settings.
-Input Player and Teams: Enter the details of participants and form teams.
-Assign Individuals to Events: Allocate participants to their respective events.
-Manage Teams: Adjust team compositions as needed.
-Enter Rankings: Input event results to generate up-to-date rankings.
-View Results: Review detailed results for both individual and team events.
